# Task 3: SQL for Data Analysis

## Objective
To use SQL queries to extract and analyze data from an Ecommerce database.

## Tools Used
MySQL Workbench

## Dataset
Ecommerce_SQL_Database

## SQL Operations Performed
- SELECT
- WHERE
- ORDER BY
- GROUP BY
- INNER JOIN
- LEFT JOIN
- RIGHT JOIN
- Subquery
- Aggregate Functions (SUM, AVG)
- VIEW Creation
- INDEX Creation

## Outcome
Learned to manipulate and query structured data using SQL.
